<Macro id="iam-requirements" />
 
You can select the version of your Redis™ database engine upon the creation of your Database Instance. Once your Database Instance is created, you can update the database engine to a newer version at any time. 
 
<Message type="requirement">
  - You have an account and are logged into the [Scaleway console](https://console.scaleway.com)
  - You have a [Redis™ Database Instance](/managed-databases/redis/how-to/create-a-database-for-redis)
</Message>

1. Click **Redis™** under **Managed Databases** on the side menu. A list of your Database Instances displays.
    <Lightbox src="scaleway-redis-engine-version-upgrade.webp" alt="" />
2. Click **Upgrade** for the database you wish to upgrade the Redis™ database engine for. A pop-up appears.
    <Message type="tip">
      Alternatively you can click the database name or <Icon name="more" /> > **More info**, then click **Upgrade** under **Version** in the Database Instances information page.
    </Message>

    <Lightbox src="scaleway-engine-version-upgrade-pop-up.webp" alt="" />
3. Select the version to which you want to upgrade. 
4. Click **Upgrade**. 
    <Message type="note">
      It is not possible to downgrade your Redis™ engine version once it has been upgraded. 
    </Message>
